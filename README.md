# 🎯 Number Guessing Game

A simple command-line number guessing game written in Python.

## 📖 About

The program generates a random number between 1 and 20, and challenges you to guess it within 3 attempts. Input validation is handled gracefully, so entering non-numeric values won't crash the game.

## 🚀 How to Run

Make sure you have Python installed, then run:
```bash
python guessing_game.py
```

## 🕹️ How to Play

1. Run the script
2. Enter a number between 1 and 20 when prompted
3. The game will tell you if you guessed correctly or how many attempts you have left
4. You have **3 attempts** to guess the correct number

## 📂 Features

- Random number generation using Python's `random` module
- Input validation — handles non-numeric input without crashing
- Clean feedback after each guess
- Win/loss messages at the end of each game

## 🛠️ Built With

- Python 3
- `random` (standard library)
